To export as PNG sprites:
  1. Open this file in Chrome
  2. Right-click a logo → Inspect → Select the .logo-container element
  3. Right-click → "Capture node screenshot"
  4. Or use a tool like html2canvas or Puppeteer for batch export

Note: The "Great Vibes" Google Font is similar to the original brush script. For an exact match, you'd need to identify the original font (possibly Bromello, Angeline, or a custom font).
